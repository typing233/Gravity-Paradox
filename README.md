# Gravity Paradox - 重力悖论

一款结合相反重力与镜像操作的双屏俄罗斯方块变体游戏，极其考验玩家的左右互搏协调能力与全局动态策略。

## 游戏特性

### 🎮 双屏联动设计
- **左右独立游戏区域**：两个完全独立的俄罗斯方块游戏场
- **相反重力系统**：左侧区域方块向下坠落，右侧区域方块向上"坠落"

### 🔄 镜像同步控制
- **单套键盘指令**：使用一套键盘同时控制两侧方块
- **镜像移动**：
  - 左箭头键：左侧方块向右移动 / 右侧方块向左移动
  - 右箭头键：左侧方块向左移动 / 右侧方块向右移动
- **镜像旋转**：两侧方块呈镜像关系旋转

### 💥 跨区连带判定
- **障碍块生成**：单侧消除行会在另一侧随机生成障碍块
- **裂纹特效**：障碍块带有半透明裂纹视觉效果
- **双重消除**：障碍块需要被消除两次才能完全清除

### ⚡ 同步消除机制
- **0.5秒窗口**：高精度时间监听，检测0.5秒内的双侧消除
- **全屏清空奖励**：成功触发同步消除可清空两侧所有方块
- **高分奖励**：同步消除奖励10000分

### 🎨 视觉设计
- **左蓝右红对比色调**：鲜明的色彩区分，便于视觉追踪
- **动态视觉特效**：
  - 行消除闪光动画
  - 障碍块裂纹效果
  - 同步消除全屏特效
  - 幽灵方块预览

## 操作说明

| 按键 | 功能 |
|------|------|
| `←` | 左侧方块右移 / 右侧方块左移 |
| `→` | 左侧方块左移 / 右侧方块右移 |
| `↑` | 旋转（镜像） |
| `↓` | 加速下落 |
| `空格` | 瞬间下落 |
| `P` | 暂停游戏 |
| `R` | 重新开始 |

## 计分规则

- **单一行消除**：100分 × 等级
- **双行消除**：300分 × 等级
- **三行消除**：500分 × 等级
- **四行消除**：800分 × 等级
- **瞬间下落**：每格2分
- **同步消除**：10000分 + 全屏清空

## 等级系统

- 每消除10行提升1级
- 等级越高，方块下落速度越快
- 最高速度限制：每100毫秒一格

## 游戏策略

### 基础操作
由于两侧控制是镜像关系，需要重新建立空间直觉：
- 想象左侧是"正常"俄罗斯方块
- 想象右侧是"倒立"的俄罗斯方块，控制方向相反

### 高级技巧
1. **同步消除练习**：有意识地在0.5秒内让两侧都消除行，触发全屏清空奖励
2. **障碍块管理**：注意控制障碍块的位置，避免堆积过高
3. **左右平衡**：保持两侧高度相对平衡，不要让一侧过快堆积
4. **预判镜像**：提前思考移动一个方块会对另一侧产生什么影响

## 技术实现

- **纯前端开发**：HTML5 + CSS3 + JavaScript (ES6+)
- **Canvas渲染**：使用HTML5 Canvas进行高性能游戏渲染
- **事件驱动**：基于键盘事件的即时响应
- **动画系统**：requestAnimationFrame实现流畅动画

## 开发理念

这款游戏的设计核心在于**认知挑战**：
- 打破传统俄罗斯方块的单一重力直觉
- 训练大脑同时处理两个相反方向的空间思维
- 引入同步消除机制，鼓励策略性思考而非单纯反应
